# Modbus-Tool

#### 介绍
This is an Android project for a ModBus Tool

 **这是一个关于Moudbus的安卓项目 **
- 采用Composable组件API开发，支持动态添加变量
- 通过TCP去访问PLC设备，批量读取、单个读取、循环读取以及类型自动转换
- 支持添加需要记录/修改的变量，软件会自动展示到界面上

#### 软件架构
Kotlin
